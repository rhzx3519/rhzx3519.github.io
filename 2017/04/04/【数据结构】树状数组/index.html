结构定义

数组A[N]的索引从1开始
Tree[i]存储了数组A[i]从i-lowbit(i)+1 到 i的和

接口实现

  1. 给A[i]加上一个数

(1) 当i > n时，算法结束，否则转到第(2)歩
(2) Tree[i] += x, i += lowbit(i), 转第(1)歩

  1. 求A[1]到A[i]的和

(1) 令sum=0，转第(2)步
(2) 加入i <= 0，算法结束，返回sun的值，否则 sum += Tree[i]，转第(3)步
(3) i-=lowbit(i)，转第(2)步

//求A[1]到A[X]的和
int get(int x) {
    int sum = 0;
    for (int i = x; i; i -= lowbit(i))
        sum += Tree[i];
    return sum;
}
