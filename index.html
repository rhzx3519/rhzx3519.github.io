1.1 зӣёжңә

еӣҫеғҸжҲҗеғҸеңЁиҝ‘е№ійқўдёҠпјҢеҸӘжңүеңЁзңјзқӣе’Ңиҝ‘е№ійқўдёҠд№Ӣй—ҙзҡ„е…үзәҝеҜ№еӣҫеғҸжңүдҪңз”ЁгҖӮ

1.2 е…үзәҝпјҚзү©дҪ“ зӣёдәӨ

е…үзәҝrзҡ„еҸӮж•°иЎЁиҫҫејҸеҰӮдёӢпјҡ

$$ r(t) = o + td, $$

е…¶дёӯпјҢoиЎЁзӨәеҺҹзӮ№пјҢdжҳҜж–№еҗ‘еҗ‘йҮҸпјҢtзҡ„е®ҡд№үеҹҹжҳҜ$[0, \infty]$пјҢеҜ№дәҺдёҖдёӘз”ұ$F(x, y, z) = 0$е®ҡд№үзҡ„иЎЁйқўпјҢжҲ‘еҸӘиҰҒе°Ҷе…үзәҝиЎЁиҫҫејҸеёҰе…ҘеҲ°ж–№зЁӢдёӯпјҢжұӮеҫ—жңҖе°Ҹзҡ„tе®һж №пјҢеҚіеҸҜеҸ–еҫ—зӣёдәӨзӮ№гҖӮдҫӢеҰӮзү©дҪ“иЎЁйқўжҳҜдёҖдёӘеңҶ:
$$ x^2 + y^2 + z^2 - r^2 = 0, $$

еёҰе…Ҙе…үзәҝж–№зЁӢпјҡ

$$ (o + td)^2 + (o + td)^2 + (o + td)^2 - r^2 = 0. $$

1.3 е…үз…§еҲҶеёғ

еҹәдәҺзү©зҗҶзҡ„жёІжҹ“йҖҡеёёеҹәдәҺеҢәеҹҹе…үжәҗпјҢеңЁеҫ®еҲҶеҢәеҹҹdAдёҠзҡ„е…үз…§иғҪйҮҸжҖ»йҮҸ$d\Phi$зҡ„иЎЁиҫҫејҸеҰӮдёӢпјҡ
$$ dE = \frac {\Phi cos\theta} {4\pi r^2}. $$
е…¶дёӯпјҢ$\Phi$иЎЁзӨәзү©дҪ“иЎЁйқўдёҖзӮ№зҡ„жҖ»иғҪйҮҸпјҢе®ғеғҸжүҖжңүж–№еҗ‘еҸ‘е°„е…үзәҝпјҢеҲҷеңЁеҢ…еӣҙзҗғйқўдёҠзҡ„иғҪйҮҸжҖ»е’Ңдёә$\Phi / (4\pi)$пјҢзҗғйқўдёҠдёҖзӮ№зҡ„иғҪйҮҸеҚ зҗғйқўдёҠжүҖжңүиғҪйҮҸзҡ„жҜ”дҫӢдёә$1/r^2$пјҢзҗғйқўдёҠдёҖеқ—еҖҫж–ңи§’пјҲиЎЁйқўжі•зәҝдёҺе…Ҙе°„е…үзәҝд№Ӣй—ҙзҡ„еӨ№и§’пјүдёә$/theta$зҡ„еҫ®еҲҶеҢәеҹҹdAпјҢжүҖеҚ иғҪйҮҸжҜ”дҫӢдёә$cos\theta$гҖӮ

2.4 дё»жёІжҹ“еҫӘзҺҜ

Sceneе’ҢRendererеҲӣе»әе№¶еҲқе§ӢеҢ–д№ӢеҗҺпјҢRenderer::Render()еҮҪж•°иў«и°ғз”ЁпјҢејҖе§Ӣpbrtжү§иЎҢзҡ„з¬¬дәҢдёӘйҳ¶ж®өпјҡдё»жёІжҹ“еҫӘзҺҜгҖӮеңЁSampleRenderзұ»зҡ„иҜҘж–№жі•зҡ„е…·дҪ“е®һзҺ°дёӯпјҢеңЁеӣҫеғҸе№ійқўдёҠзҡ„жҜҸдёӘдҪҚзҪ®пјҢиҜҘж–№жі•дҪҝз”ЁCameraе’ҢSamplerз”ҹжҲҗдёҖжқЎе°„е…ҘеңәжҷҜдёӯзҡ„е…үзәҝпјҢжҺҘзқҖдҪҝз”Ёз§ҜеҲҶеҷЁеҫ—еҲ°еҲ°иҫҫеӣҫеғҸе№ійқўзҡ„жІҝзқҖиҜҘе…үзәҝзҡ„е…үж•°йҮҸпјҢиҝҷдёӘеҖјиў«дј з»ҷFilmпјҢи®Ўз®—е…үзәҝзҡ„иҙЎзҢ®гҖӮ

еңЁжёІжҹ“ејҖе§Ӣд№ӢеүҚпјҢRenderи°ғз”ЁSamplerRenderer:Preprocess()ж–№жі•пјҢз”ЁжқҘеҲқе§ӢеҢ–дёҖдәӣзү№ж®Ҡе…үжәҗжҲ–иҖ…еӨ„зҗҶеңәжҷҜдёӯдёҖдәӣз”ЁдёҚеҲ°зҡ„еҮ дҪ•дҪ“пјҢ

еңЁжёІжҹ“ејҖе§Ӣд№ӢеүҚпјҢRenderд№ҹдјҡеҲӣе»әдёҖдёӘSampleеҜ№иұЎпјҢSamplerдјҡе°ҶеңЁдё»еҫӘзҺҜдёӯз”ҹжҲҗзҡ„ж ·жң¬еҖјеӯҳеӮЁеҲ°е®ғйҮҢйқўгҖӮ

дё»жёІжҹ“еҫӘзҺҜејҖе§Ӣд№ӢеҗҺпјҢз”ҹжҲҗеӣҫеғҸзҡ„д»»еҠЎиў«еҲҶй…Қз»ҷдёҖзі»еҲ—зҡ„SamplerRendererTasksпјҢжҜҸдёӘSamplerRendererTasksиҙҹиҙЈеӣҫзүҮзҡ„дёҖе°Ҹеқ—й•ҝж–№еҪўеҢәеҹҹгҖӮSamplerRendererTaskз»§жүҝиҮӘTaskжҠҪиұЎзұ»гҖӮ

EnqueueTasks()е®ҡд№үеңЁcore/parallel.hдёӯпјҢе®ғиҫ“е…ҘдёҖдёӘtasksйӣҶеҗҲпјҢз„¶еҗҺеңЁжүҖжңүеӨ„зҗҶеҷЁдёҠжү§иЎҢе®ғд»¬гҖӮд№ӢеҗҺSamplerR enderзҡ„Render()еҮҪж•°и°ғз”ЁWaitForAllTasks()ж–№жі•зӯүеҫ…жүҖжңүд»»еҠЎжү§иЎҢе®ҢжҜ•гҖӮ

SamplerRenderдҝқиҜҒжҜҸдёӘеӨ„зҗҶеҷЁиҮіе°‘жңү32дёӘд»»еҠЎеңЁжү§иЎҢпјҢеҗҢж—¶и®Ўз®—е…ұеҸҜд»ҘеҲҶжҲҗеӨҡе°‘дёӘ16x16еғҸзҙ еӨ§е°Ҹзҡ„д»»еҠЎгҖӮ

д»»еҠЎе®ҢжҲҗд№ӢеҗҺпјҢSamplerRender:render()еҲ йҷӨж ·жң¬пјҢи°ғз”ЁFilm:WriteImage()е°ҶеӣҫеғҸеҶҷе…ҘзЈҒзӣҳгҖӮ

дёӢйқўе…·дҪ“зңӢдёӢSamplerRendererTask.зҡ„е®һзҺ°пјҡ

SamplerRendererTaskзӢ¬з«ӢдәҺSamplerRender:render()иҝҗиЎҢпјҢжһ„йҖ еҮҪж•°е’ҢжҲҗе‘ҳеҸҳйҮҸеҰӮдёӢпјҡ

Taskиў«еҲҶй…ҚеҲ°зү№е®ҡзҡ„еӨ„зҗҶд№ӢеҗҺпјҢTask:run()ж–№жі•дјҡиў«и°ғз”ЁпјҢйҰ–е…Ҳдјҡи®ҫзҪ®дёҖдәӣиҝҗиЎҢж•°жҚ®пјҢеҶіе®ҡиҙҹиҙЈFilmе№ійқўзҡ„е“ӘдёҖеқ—еҢәеҹҹпјҢеңЁйҮҮж ·еҷЁз”ҹжҲҗж ·жң¬д№ӢеүҚеҲҶй…ҚеҶ…еӯҳпјҢеҶіе®ҡзҰ»ејҖзӣёжңәзҡ„зӣёеә”зҡ„е…үзәҝпјҢеҶіе®ҡз”ЁдәҺи®Ўз®—еҲ°иҫҫFimе№ійқўзҡ„е…үзәҝзҡ„з§ҜеҲҶеҷЁгҖӮ

SamplerRendererTaskдҪҝз”ЁSampler::GetSubSampler()еҫ—еҲ°дёҖдёӘеҸӘиҙҹиҙЈз”ҹжҲҗеӣҫеғҸеӯҗеқ—ж ·жң¬зҡ„ж–°ж ·жң¬еҷЁпјҢеҰӮжһңиҝ”еӣһз©әпјҢеҲҷSamplerRendererTask:run()еҮҪж•°иҝ”еӣһгҖӮ

SamplerRendererTask:run()еҮҪж•°дёӯиҝҳжңүдёӨдёӘжң¬ең°еҸҳйҮҸMemoryArenaе’ҢRNGпјҢMemoryArenaиў«дј з»ҷз§ҜеҲҶеҷЁпјҢиҙҹиҙЈе°ҸеҶ…еӯҳеҲҶй…ҚпјҢRNGз”ЁдәҺеңЁз§ҜеҲҶеҷЁдёӯз”ҹжҲҗmonte carloйҮҮж ·жүҖйңҖзҡ„дјӘйҡҸжңәж•°гҖӮ

йҮҮж ·еҷЁдә§з”ҹзҡ„ж ·жң¬ж•°йҮҸжҳҜйҡҸжңәзҡ„пјҢеҜ№дәҺдёҖдәӣж ·жң¬з”ҹжҲҗз®—жі•пјҢдёҖж¬Ўз”ҹжҲҗжүҖжңүж ·жң¬еҖјпјҢзӣёжҜ”дәҺдёҖж¬Ўз”ҹжҲҗдёҖдёӘж ·жң¬еҖјжӣҙеҠ з®ҖеҚ•гҖӮSampler::MaximumSampleCount()иҝ”еӣһж ·жң¬еҷЁдёҖж¬ЎиғҪеӨҹз”ҹжҲҗзҡ„ж ·жң¬еҖјзҡ„дёҠйҷҗгҖӮж №жҚ®иҝҷдёӘдёҠйҷҗеҖјпјҢж ·жң¬еҲӣе»әе№¶еӯҳеӮЁзӣёеә”ж•°йҮҸзҡ„ж•°жҚ®з»“жһ„гҖӮеҜ№дәҺжҜҸдёӘж ·жң¬пјҢжңүеҝ…иҰҒеӯҳеӮЁзӣёеә”зҡ„е…үзәҝж•°жҚ®(RayDifferential)пјҢжІҝзқҖе…үзәҝзҡ„е…үи°ұLsпјҢжІҝзқҖе…үзәҝзҡ„йҖҸе°„зҺҮTsе’ҢдёҖдёӘе…үзәҝзү©дҪ“зӣёдәӨзӮ№зҡ„ж•°жҚ®з»“жһ„IntersectionгҖӮ

жҜҸж¬ЎеҫӘзҺҜпјҢSampler::GetMoreSamples()йғҪдјҡиў«и°ғз”ЁеҲқе§ӢеҢ–ж ·жң¬йӣҶеҗҲпјҢиҝҷдёӘеҮҪж•°иҝ”еӣһеҲқе§ӢеҢ–зҡ„ж ·жң¬ж•°йҮҸгҖӮжҺҘдёӢжқҘзҡ„д»Јз ҒдјҡдёәжҜҸдёӘж ·жң¬жүҫеҲ°еҜ№еә”зҡ„зӣёжңәеҮәе°„е…үзәҝпјҢе°Ҷе…үзәҝдј еҲ°з§ҜеҲҶеҷЁи®Ўз®—еҲ°иҫҫFilmе№ійқўзҡ„е…үзәҝпјҢжңҖеҗҺе°Ҷж ·жң¬зҡ„иҙЎзҢ®еҖјеҠ е…ҘеҲ°еӣҫеғҸдёӯпјҢйҮҠж”ҫеҶ…еӯҳгҖӮ

Cameraзұ»жҸҗдҫӣдәҶдёӨдёӘж–№жі•пјҡ

  1. Camera::GenerateRay(): иҝ”еӣһеӣҫеғҸйҮҮж ·дҪҚзҪ®зҡ„е…үзәҝRay
  2. Camera::GenerateRayDifferential(): иҝ”еӣһRayDifferentialпјҢеҢ…еҗ«дәҶйҮҮж ·дҪҚзҪ®е‘Ёеӣҙ1дёӘеғҸзҙ зҡ„е…үзәҝдҝЎжҒҜ

е…үзәҝеҫ®еҲҶдҝЎжҒҜиғҪеӨҹи®©дёҖдәӣзә№зҗҶеҮҪж•°еҫ—еҲ°жӣҙеҘҪзҡ„з»“жһңпјҢиғҪеӨҹжӣҙеҘҪең°и®Ўз®—зә№зҗҶеңЁеғҸзҙ з©әй—ҙзҡ„еҸҳеҢ–пјҢжҳҜеҸҚиө°ж ·зҡ„е…ій”®йғЁеҲҶгҖӮRayDifferentialз»§жүҝиҮӘRayпјҢд»–жңүдёӨдёӘйўқеӨ–зҡ„жҲҗе‘ҳеҸҳйҮҸRays rxе’ҢryпјҢеӯҳеӮЁзӣёйӮ»зҡ„е…үзәҝдҝЎжҒҜгҖӮ

Camera:GenerateRayDifferential()еҮҪж•°иҝ”еӣһе…үзәҝзҡ„жө®зӮ№жқғйҮҚгҖӮеҜ№дәҺз®ҖеҚ•зҡ„зӣёжңәжЁЎеһӢпјҢжҜҸжқЎе…үзәҝзҡ„жқғйҮҚйғҪжҳҜдёҖж ·зҡ„пјҢеҜ№дәҺеӨҚжқӮзҡ„еҰӮй•ңзүҮжһ„жҲҗзҡ„зӣёжңәжЁЎеһӢпјҢжқғйҮҚеҖјеҸҜиғҪдјҡдёҚдёҖж ·гҖӮиҝҷз§ҚзӣёжңәжЁЎеһӢиғҪеӨҹжЁЎжӢҹе°‘йҮҸе…үеҲ°иҫҫеӣҫеғҸиҫ№зјҳпјҢиҖҢеӨ§йҮҸе…үйӣҶдёӯеңЁеӣҫеғҸдёӯеҝғзҡ„ж•ҲжһңпјҢиҝҷз§Қж•ҲжһңеҸ«еҒҡе…үжҷ•гҖӮ

зҺ°еңЁжҲ‘д»¬жңүдәҶдёҖжқЎе…үзәҝпјҢжҺҘдёӢжқҘжҳҜзЎ®е®ҡжІҝзқҖйӮЈжқЎе…үзәҝеҲ°иҫҫеӣҫеғҸе№ійқўзҡ„е…үзҡ„ж•°йҮҸпјҲе®ғзҡ„иҫҗе°„пјүгҖӮжҲ‘д»¬з”ЁLд»ЈиЎЁе…үзәҝзҡ„иҫҗе°„пјҢи®Ўз®—е…үзәҝиҫҗе°„зҡ„ж–№жі•жҳҜRenderer::Li()пјҢиҫҗе°„еҖјдҪҝз”ЁSpectrumзұ»д»ЈиЎЁпјҢиҝҷжҳҜpbrtзі»з»ҹз”ЁдәҺжҠҪиұЎдёҚеҗҢжіўй•ҝдёҠиғҪйҮҸеҲҶеёғзҡ„зұ»пјҚжҚўеҸҘиҜқиҜҙпјҢе°ұжҳҜйўңиүІгҖӮ

еӨ„зҗҶе®ҢдёҖжү№ж ·жң¬д№ӢеҗҺпјҢжүҖжңүMemoryArenaеҲҶй…Қзҡ„еҶ…еӯҳйғҪдјҡиў«йҮҠж”ҫпјҢ

SamplerRenderer::Li()еңЁSamplerRendererTask::Run()еҮҪж•°дёӯиў«з”ЁдәҺи®Ўз®—е…үзәҝиҫҗе°„еҖјгҖӮScene::Intersect()з”ЁдәҺи®Ўз®—е…үзәҝдёҺеңәжҷҜдёӯеӣҫе…ғзҡ„з¬¬дёҖдёӘзӣёдәӨзӮ№пјҢеҰӮжһңеӯҳеңЁзӣёдәӨзӮ№пјҢSamplerRenderer::Li()и°ғз”ЁSurfaceIntegrator::Li()и®Ўз®—зӣёдәӨзӮ№зҡ„еҮәе°„е…үзәҝ$L_o$пјҢе°Ҷз»“жһңеӯҳеӮЁеҲ°$L_i$гҖӮжҺҘзқҖи°ғз”ЁVolumeIntegrator::Transmittance()е…үзәҝеӣ дёәеҸӮдёҺеӘ’д»ӢеңЁзӣёжңәе’ҢзӣёдәӨзӮ№д№Ӣй—ҙзҡ„иЎ°еҮҸеҖјгҖӮжңҖеҗҺпјҢVolumeIntegrator::Li()зЎ®е®ҡдәҶ$L_v$иў«еҠ е…ҘеҲ°иҫҗе°„еҖјдёӯгҖӮиҝҷдәӣзӣёдә’дҪңз”Ёзҡ„еҮҖж•ҲжһңжҳҜ$TL_i + L_v$пјҢе…·дҪ“и§ЈйҮҠеңЁ16.1з« пјҢ

еҜ№дәҺжІЎжңүеҮ»дёӯд»»дҪ•еӣҫе…ғзҡ„е…үзәҝпјҢLight::LeеҮҪж•°иў«з”ЁдәҺи®Ўз®—иҫҗе°„еҖјпјҢ

SamplerRenderer::Transmittance()з”ЁдәҺзӢ¬з«Ӣи®Ўз®—е…үзәҝиЎ°еҮҸеҖј

SurfaceIntegrator::Li()жҳҜжүҖжңүиЎЁйқўз§ҜеҲҶеҷЁзҡ„е…ій”®еҮҪж•°пјҢе®ғиҝ”еӣһе…үзәҝзҡ„иҫҗе°„еҖјпјҢ

Whitted з§ҜеҲҶеҷЁдјҡжІҝзқҖеҸҚе°„е’ҢжҠҳе°„зҡ„е…үзәҝйҖ’еҪ’жұӮеҸ–иҫҗе°„еәҰпјҢе®ғдјҡеңЁеҲ°иҫҫжңҖеӨ§ж·ұеәҰж—¶еҒңжӯўйҖ’еҪ’WhittedIntegrator::maxDepthпјҢй»ҳи®ӨжңҖеӨ§ж·ұеәҰжҳҜ5пјҢ

BSDF(bidirectional scattering distribution functions)з”ЁжқҘжҸҸиҝ°зү©дҪ“иЎЁйқўеҰӮдҪ•еҸҚе°„е…үзәҝпјҢеңЁpbrtзі»з»ҹдёӯжҲ‘д»¬дҪҝз”ЁBSDFзұ»иЎЁзӨәгҖӮpbrtжҸҗдҫӣдәҶеӣҫеҪўеӯҰдёӯеёёз”Ёзҡ„еҮ з§Қж ҮеҮҶж•Је°„еҮҪж•°пјҢжҜ”еҰӮLambertianеҸҚе°„е’ҢTorranceвҖ“Sparrowеҫ®е°ҸйқўзүҮжЁЎеһӢгҖӮBRDFдјҡеңЁз¬¬е…«з« и®Ёи®әгҖӮ

дёәдәҶеҫ—еҲ°зӣёдәӨзӮ№зҡ„BSDFпјҢз§ҜеҲҶеҷЁдјҡи°ғз”ЁIntersection::GetBSDF()еҮҪж•°пјҡ

дёҠеӣҫжҸҸиҝ°дәҶзі»з»ҹдёӯеёёз”Ёзҡ„дёҖдәӣеҹәжң¬жҖ§иҙЁпјҡpд»ЈиЎЁзӣёдәӨзӮ№пјҢnжҳҜзӣёи§ҒзӮ№еҫ®еҲҶиЎЁйқўзҡ„жі•зәҝпјҢд»ҺpзӮ№жҢҮеҗ‘е…үзәҝеҺҹзӮ№зҡ„еҗ‘йҮҸдёә$\omega_o$пјӣеҚ•дҪҚеҢ–зҡ„ж–№еҗ‘еҗ‘йҮҸеңЁжң¬д№ҰдёӯдҪҝз”Ё$\omega$иЎЁзӨәпјҢеңЁpbrtзҡ„д»Јз ҒдёӯжҲ‘д»¬дјҡдҪҝз”Ёз®ҖеҶҷwoд»ЈиЎЁ$\omega_o$пјҢд»ЈиЎЁеҮәе°„е…үзәҝгҖӮ

еҪ“е…үзәҝйӣҶдёӯдёҖдёӘеҸ‘е…үдҪ“ж—¶пјҢз§ҜеҲҶеҷЁйҖҡиҝҮи°ғз”ЁIntersection::Le()еҮҪж•°и®Ўз®—еҸ‘е…үдҪ“еҸ‘е°„зҡ„иҫҗе°„еәҰпјҢиҝҷз»ҷеҮәдәҶе…үзәҝдј йҖ’зӯүејҸзҡ„йҰ–йЎ№пјҢеҰӮжһңзү©дҪ“дёҚжҳҜеҸ‘е…үдҪ“пјҢеҲҡеҮҪж•°иҝ”еӣһдёҖдёӘй»‘иүІзҡ„е…үи°ұпјҲеҚійўңиүІпјүгҖӮ

$$
L_0(p, \omega_o) = L_e(p, \omegao) + \int\xi^2 f(p, \omega_o, \omega_i)L_i(p, \omega_i)\mid cos\theta_i \mid d \omega_i
$$

еҜ№дәҺжҜҸдёҖжқЎе…үзәҝпјҢз§ҜеҲҶеҷЁи°ғз”ЁLight::Sample_L()и®Ўз®—зқҖиүІзӮ№иҫҗе°„еәҰпјҢиҝҷдёӘеҮҪж•°еҗҢж ·иҝ”еӣһд»ҺзқҖиүІзӮ№жҢҮеҗ‘е…үжәҗзҡ„ж–№еҗ‘еҗ‘йҮҸwi($\omega_i$)гҖӮ

Sample_L()еҮҪж•°еҗҢж ·е…үзәҝеңЁwiж–№еҗ‘иў«йҮҮж ·зҡ„жҰӮзҺҮеҜҶеәҰпјҢеӯҳеӮЁеңЁpdfеҸҳйҮҸдёӯгҖӮиҝҷдёӘеҖјиў«з”ЁдәҺеӨҚжқӮеҢәеҹҹе…үжәҗзҡ„monte carloз§ҜеҲҶи®Ўз®—дёӯпјҢе…үзәҝд»Һи®ёеӨҡж–№еҗ‘еҲ°иҫҫзқҖиүІзӮ№пјҢеҚҙеҸӘжңүдёҖдёӘж–№еҗ‘иў«йҮҮж ·гҖӮеҜ№дәҺзӮ№е…үжәҗpdfзҡ„еҖјжҳҜ1пјҢжҲ‘д»¬дјҡеңЁз¬¬13з« еҲ°з¬¬15з« и®Ёи®әз»ҶиҠӮгҖӮ

еҰӮжһңеҲ°иҫҫзҡ„зқҖиүІзӮ№зҡ„иҫҗе°„еәҰдёҚдёә0пјҢеҗҢж—¶BSDFеҮҪж•°жҳҫзӨәжңүдёҖдәӣе…Ҙе°„е…үзәҝwiиў«ж•Је°„еҲ°еҮәиүІе…үзәҝwoпјҢз§ҜеҲҶеҷЁе°ұдјҡе°ҶBSDF fд№ҳдёҠиҫҗе°„еәҰ$L_i$пјӣcosйЎ№(йҖҡиҝҮ$w_i$е’ҢиЎЁйқўжі•еҗ‘йҮҸnзҡ„зӮ№д№ҳеҫ—еҲ°)пјӣзқҖиүІзӮ№е’Ңе…үжәҗд№Ӣй—ҙзҡ„йҖҸе°„зҺҮгҖӮиҝҷдёӘд№ҳз§Ҝд»ЈиЎЁе…үзәҝеҜ№е…үзәҝдј йҖ’зӯүејҸз§ҜеҲҶзҡ„иҙЎзҢ®еҖјпјҢе®ғиў«еҠ е…ҘеҲ°еҸҚе°„иҫҗе°„еәҰзҡ„жҖ»е’Ң$L_o$дёӯгҖӮеңЁжүҖжңүе…үзәҝйғҪиў«иҖғиҷ‘д№ӢеҗҺпјҢз§ҜеҲҶеҷЁи®Ўз®—дәҶзӣҙжҺҘе…үзәҝпјҲд»Һе…үжәҗзӣҙжҺҘеҲ°иҫҫзү©дҪ“иЎЁйқўзҡ„е…үзәҝпјҢеҸҰеӨ–дёҖз§Қй—ҙжҺҘе…үзәҝжҳҜд»Һе…¶д»–зү©дҪ“еҸҚе°„иҝҮжқҘзҡ„пјүжҖ»иҙЎзҢ®еҖјгҖӮ

L_0(p, \omega_o) = L_e(p, \omegao) + \int\xi^2 f(p, \omega_o, \omega_i)L_i(p, \omega_i)\mid cos\theta_i \mid d \omega_i

Rendering Equation

$$
L_r(\overrightarrow x, \overrightarrow \omega) = \int_A f_r(\overrightarrow x, \overrightarrow \omega(\overrightarrow x, \overrightarrow xвҖҷ), \overrightarrow \omega L_0(\overrightarrow x,\overrightarrow \omega(\overrightarrow xвҖҷ, \overrightarrow x)) G(\overrightarrow x, \overrightarrow xвҖҷ) V(\overrightarrow x, \overrightarrow xвҖҷ)dA(\overrightarrow xвҖҷ))
$$

where
$$ G(\overrightarrow x, \overrightarrow xвҖҷ) = G(\overrightarrow xвҖҷ, \overrightarrow x) = \frac {cos\theta_i cos\theta_0вҖҷ} {|\overrightarrow x - \overrightarrow xвҖҷ|^2} V(\overrightarrow x, \overrightarrow xвҖҷ) $$
дёҠйқўзҡ„е…¬ејҸеҸҜд»Ҙз®ҖеҶҷдёәпјҡ

$$ L = L_e + K \circ L $$

KеҸҜд»ҘеҲҶи§ЈдёәTе’ҢSпјҢTд»ЈиЎЁе…үзәҝдј йҖ’пјҢе®ғе°ҶдёҖдёӘиЎЁйқўзҡ„еҮәе°„е…үзәҝиҪ¬жҚўдёәеҸҰдёҖиЎЁйқўзҡ„е…Ҙе°„е…үзәҝ: $$ L_i(\overrightarrow x, \overrightarrow \omega(\overrightarrow xвҖҷ, \overrightarrow x)) = T \circ L(\overrightarrow x, \overrightarrow \omega(\overrightarrow x, \overrightarrow xвҖҷ)) $$

Sд»ЈиЎЁеҸҚе°„жҲ–иҖ…ж•Је°„зҡ„е…үзәҝпјҢе®ғж №жҚ®е…Ҙе°„е…үзәҝи®Ўз®—еҮәеҮәе°„е…үзәҝпјҡ $$ L_r(\overrightarrow x, \overrightarrow \omega) = S \circ L_i(\overrightarrow x, \overrightarrow \omegaвҖҷ) $$

rendering equationеҸҜд»ҘйҖҡиҝҮд»ҘдёӢзҡ„иҝӯд»Јж–№ејҸжұӮеҮәе®ғзҡ„и§Јпјҡ $$ L^0 = L_e $$
$$ L^1 = L_e + K \circ L^0 = L_e + K \circ L_e $$
$$ вҖҰ $$
$$ L^n = Le + K \circ L^{n-1} = \sum{i = 0}^nK^n \circ L_e $$

е…¶дёӯ$K^0 = I$, IжҳҜеҚ•дҪҚж“ҚдҪңз¬ҰпјҢдёҠиҝ°ж— з©·жұӮе’ҢзӯүејҸиў«з§°дёәзәҪжӣјзә§ж•°(Neumann Series)пјҢе®ғеұ•зӨәдәҶrendering equationзҡ„еҪўејҸи§Ј.
еҸҰдёҖз§Қи§ЈйҮҠзәҪжӣјзә§ж•°зҡ„ж–№ејҸжҳҜпјҡ
$$ \frac 1 {1-x} = (1 - x)^{-1} = 1 + x + x^2 вҖҰ, $$
and
$$ (1 - K)^{-1} = I + K + K^2 вҖҰ, $$
еҲҷrendering equation жңүд»ҘдёӢи§Ј:
$$ L = (1 - K)^{-1}L_e $$

д№ҹеҸҜд»ҘеҶҷдҪңпјҡ
$$ L(\overrightarrow x, \overrightarrow \omega) = \sum_{i=0}^\infty K^i \circ L_e(\overrightarrow x_0, \overrightarrow \omega_0) $$

иҜҘзӯүејҸиҝҳжҳҜзӣёеҪ“зҡ„еӨҚжқӮе•ҠпјҢжҲ‘д»¬жҺҘдёӢжқҘдјҡз”Ёmonte carlo ж–№жі•жұӮи§ЈиҜҘж–№зЁӢзҡ„и§ЈгҖӮ

е°Ҷrendering equationе’Ңзӣёжңәдёӯзҡ„еӣҫеғҸеҪўжҲҗиҝҮзЁӢиҒ”зі»иө·жқҘпјҢжҲ‘д»¬дјҡеҫ—еҲ°дёҖдёӘжӣҙеҠ жңүз”Ёзҡ„зӯүејҸMeasurement Equation:
$$
M = \intA \int\Omega \int_T R(\overrightarrow x, \overrightarrow \omega, t)L(\overrightarrow x, \overrightarrow \omega,t) dt d\omega dA
$$

е…¶дёӯ

$ R(\overrightarrow x, \overrightarrow \omega, t) $ иў«з§°дёәеҸҚеә”еҮҪж•°(Response Function)пјҢе®ғзҡ„дёүдёӘеҸҳйҮҸеҲҶеҲ«д»ЈиЎЁеғҸзҙ иҝҮж»ӨеҷЁ($\overrightarrow x$)пјҢе…үеңҲ($ \overrightarrow \omega $)пјҢ еҝ«й—Ё(t)гҖӮ
еҰӮдёҠжүҖиҝ°пјҢеӣҫеғҸдёӯзҡ„еғҸзҙ еҖјжҳҜдёҖдёӘеӨҡйҮҚз§ҜеҲҶеҮҪж•°пјҢиҝҷдәӣз§ҜеҲҶйқһеёёеӨҚжқӮпјҢдҪҶжҳҜжҲ‘д»¬еҸҜд»Ҙйқһеёёе®№жҳ“ең°жұӮеҮәиў«з§ҜеҮҪж•°пјҢйҖҡиҝҮйҮҮж ·еҜ№еә”зҡ„еҮҪж•°гҖӮ

  • еңЁ(x, y)дёҠйҮҮж ·еғҸзҙ еҸҜд»Ҙйў„иҝҮж»ӨеӣҫеғҸпјҢеҮҸе°‘иө°ж ·гҖӮ
  • еңЁ(u, v)дёҠйҮҮж ·зӣёжңәзҡ„е…үеңҲеҸҜд»Ҙз”ҹжҲҗжҷҜж·ұпјҲfield of depthпјү

  • еңЁе…үзәҝзҡ„жіўй•ҝ$\lambda$дёҠйҮҮж ·еҸҜд»Ҙз”ҹжҲҗиҜёеҰӮејҘж•Јзҡ„е…үи°ұж•Ҳжһң

$$ P_j^1 = p_j^0 + \sumi p{i, j}P_i^0 $$
$$ вҖҰ $$

йҮҮж ·зҗҶи®ә

д»Һе®ҡд№үеңЁиҝһз»ӯеҹҹеңЁеҮҪж•°дёӯеҸ–еҫ—зҰ»ж•Јзҡ„ж ·жң¬еҖјпјҢеҲ©з”Ёиҝҷдәӣж ·жң¬еҖјйҮҚжһ„дёҖдёӘдёҺеҺҹеҮҪж•°иҝ‘дјјзҡ„еҮҪж•°гҖӮ

е·Ұдҫ§жҳҜеҺҹеҮҪж•°пјҢе·Ұдҫ§еҲҷжҳҜйҮҚе»әеҗҺзҡ„еҮҪж•°

йў‘еҹҹдёҺеӮ…з«ӢеҸ¶еҸҳжҚў

еӮ…з«ӢеҸ¶еҸҳжҚўе°ҶеҮҪж•°д»Һз©әй—ҙеҹҹеҸҳжҚўеҲ°йў‘еҹҹгҖӮ
дёӢеӣҫеұ•зӨәдәҶеҮҪж•°еңЁз©әй—ҙеҹҹе’Ңйў‘еҹҹдёӯдёҚеҗҢзҡ„еӣҫеҪў

еӨ§йғЁеҲҶзҡ„еҮҪж•°еҸҜд»ҘеҲҶи§ЈдёәдёҖзі»еҲ—sinеҮҪж•°зҡ„е’Ң
дёҖз»ҙеҮҪж•°$f(x)$зҡ„еӮ…з«ӢеҸ¶еҸҳжҚўеҰӮдёӢ:
$$ F(\omega) = \int_{-\infty}^{\infty}f(x)e^{-i2\pi\omega x}dx. $$

е…¶дёӯ$$e^{ix} = cosx + isinx$$
йҖҶеӮ…з«ӢеҸ¶еҸҳжҚў
$$ f(x) = \int_{-\infty}^{\infty}F(\omega)e^{i2\pi\omega x}d\omega. $$

дёӢйқўеҲ—дәҶдёҖдәӣеёёз”ЁеҮҪж•°д»ҘеҸҠеҜ№еә”зҡ„йў‘еҹҹдёӯзҡ„еҮҪж•°

Dirac delta distribution
$$ \int\delta(x)dx = 1 $$
е…¶дёӯ$еҪ“ x\neq 0, \delta(x) = 0$
$$\int f(x)\delta(x)dx = f(0).$$

йҮҮж ·е’ҢйҮҚе»ә

shahеҮҪж•°зҡ„е®ҡд№үеҰӮдёӢ:
$$ IIIT(x) = T\sum{i=-\infty}^{\infty} \delta(x - iT). $$

Tд»Јз ҒйҮҮж ·е‘ЁжңҹжҲ–иҖ…йҮҮж ·йў‘зҺҮ

$$ III_T(x)f(x) = T\sum_i \delta(x - iT)f(iT). $$

shahеҮҪж•°дёҺеҮҪж•°$f(x)$зҡ„з»“жһңжҳҜдёҖдёӘж— з©·зҡ„еҮҪж•°еҖјеәҸеҲ—
йҮҚжһ„еҮҪж•°йҖҡиҝҮд»ҘдёӢж–№зЁӢеҫ—еҲ°пјҡ

$$ \widetilde f = (III_T(x)f(x))\otimes r(x) $$

е…¶дёӯ$r(x)$жҳҜдёҖдёӘйҮҚжһ„иҝҮж»ӨеҮҪж•°пјҢ$\otimes$д»ЈиЎЁеҚ·з§Ҝ:
$$ f(x)\otimes g(x) = \int_{-\infty}^\infty f(xвҖҷ)g(x - xвҖҷ) d(xвҖҷ) $$

дёүи§’йҮҚжһ„иҝҮж»ӨеҮҪж•°(triangle reconstruction filter)дёә$f(x) = max(0, 1 - |x|).$
з®ҖеҢ–еҗҺзҡ„йҮҚжһ„еҮҪж•°еҰӮдёӢ:
$$ \widetilde f(x) = T\sum_{i=-\infty}^\infty f(iT)r(x - iT). $$
жҲ‘д»¬еҒҮи®ҫеҮҪж•°$f(x)$жҳҜеёҰйҷҗзҡ„(band limit)пјҢеҚіеӯҳеңЁйў‘зҺҮеҖј$\omega_0$еҮҪж•°$f(x)$зҡ„зҡ„жүҖжңүйў‘зҺҮеҖјйғҪе°ҸдәҺ$\omega_0$
$$ F(\omega) = 0, |\omega| = \omega_0 $$
еӮ…з«ӢеҸ¶еҸҳжҚўе…·жңүд»ҘдёӢдёҖдәӣйҮҚиҰҒзҡ„зү№жҖ§пјҡ

  1. дёӨдёӘеҮҪж•°д№ҳз§Ҝзҡ„еӮ…з«ӢеҸ¶еҸҳжҚўзӯүдәҺеҗ„иҮӘеӮ…з«ӢеҸ¶еҸҳжҚўзҡ„еҚ·з§Ҝ: $$ \mathscr F\lbrace f(x)g(x) \rbrace = F(x)\otimes G(x). $$
  2. дёӨдёӘеҮҪж•°еҚ·з§Ҝзҡ„еӮ…з«ӢеҸ¶еҸҳжҚўзӯүдәҺеӮ…з«ӢеҸ¶еҸҳжҚўзҡ„д№ҳз§Ҝ: $$ \mathscr F\lbrace \otimes f(x)g(x) \rbrace = F(x) G(x). $$
  3. дёҖдёӘе‘ЁжңҹдёәTзҡ„shahеҮҪж•°зҡ„еӮ…з«ӢеҸ¶еҸҳжҚўжҳҜдёҖдёӘе‘Ёжңҹдёә$\frac 1 T$зҡ„shahеҮҪж•°пјҢ иҝҷж„Ҹе‘ізқҖеңЁз©әй—ҙеҹҹдёӯйҮҮж ·зҡ„й—ҙйҡ”и¶Ҡй•ҝпјҢеңЁйў‘еҹҹдёӯзҡ„йҮҮж ·й—ҙйҡ”и¶Ҡе°Ҹ
  4. д»»ж„ҸеҮҪж•°дёҺdeltaеҮҪж•°зҡ„еҚ·з§ҜзӯүдәҺеҮҪж•°жң¬иә«пјҢд»»ж„ҸеҮҪж•°дёҺshahеҮҪж•°зҡ„еҚ·з§ҜзӯүдәҺеҺҹеҮҪж•°зҡ„ж— з©·еәҸеҲ—пјҢдёәдәҶиҲҚејғйў‘еҹҹдёӯйҷӨдәҶеҺҹзӮ№д№ӢеӨ–зҡ„жүҖжңүеҮҪж•°пјҢжҲ‘д»¬д№ҳд»ҘдёҖдёӘе‘ЁжңҹдёәTзҡ„зӣ’еҮҪж•°$Pi_T(x)$:
    $$
    \Pi_T(x) =
    \begin{cases}
    1 / {(2T)} |x|<T \
    0 otherwise.
    \end{cases}
    $$

зҗҶжғізҡ„йҮҮж ·е’ҢйҮҚе»әиҝҮзЁӢеҸҜд»ҘжҖ»з»“дёәеҰӮдёӢе…¬ејҸпјҡ
$$ \widetilde F = (F(\omega)\otimes III_{\frac 1 T}(\omega)) \Pi_T(x). $$

зӣёеҗҢзҡ„иҝҮзЁӢеә”з”ЁдәҺз©әй—ҙеҹҹеҗҢж ·иғҪеӨҹйҮҚе»ә$f(x)$еҮҪж•°пјҢboxеҮҪж•°зҡ„йҖҶеӮ…з«ӢеҸ¶еҸҳжҚўжҳҜ$sinc(x) = \frac {sin(\pi x)} {\pi x}$:

$$
\widetilde f = (f(x)IIIT(x)) \otimes sinc(x).
or
\widetilde f = \sum{i = -\infty}^\infty sinc(x - i)f(i).
$$

##Using Precomputed Radiance Transfer

There are several forms of complexity present in interesting scenes, including how the lighting environment is modeled (that is, area lighting models versus point/directional ones) and what kind of global effects are modeled (for example, shadows, interreflections, subsurface scattering.) Traditional interactive rendering techniques model a limited amount of this complexity. PRT enables these effects with some significant restrictions:

  • Objects are assumed to be rigid (that is, no deformations).
  • It is an object-centric approach (unless objects are moved together, these global effects are not maintained between them).
  • Only low-frequency lighting is modeled (resulting in soft shadows.) For high-frequency lights (sharp shadows), traditional techniques would have to be employed.

PRT requires either of the following, but not both:

  • highly-tessellated models and vs_1_1
  • ps_2_0

##How PRT Works

The following terms are useful for understanding how PRT works, as illustrated in the following diagram.

Source Radiance: The source radiance represents the lighting environment as a whole. In PRT an arbitrary environment is approximated using the spherical harmonic basis - this lighting is assumed to be distant relative to the object (the same assumption that is made with environment maps.)

Exit Radiance: Exit radiance is the light leaving from a point on the surface from any possible source (reflected radiance, subsurface scattering, emission).

Transfer Vectors: Transfer vectors map Source Radiance into exit radiance and are precomputed offline using a complex light transport simulation.

PRT factors the rendering process into two stages, as shown in the following diagram:

  1. An expensive light transport simulation precomputes transfer coefficients that can be used at run time.
  2. A relatively lightweight run-time stage first approximates the lighting environment using the spherical harmonic basis, then uses these lighting coefficients and the precomputed transfer coefficients (from stage 1) with a simple shader, resulting in exit radiance (the light leaving the object).
